# low-entropy-eschalot
Generates easy to read .onion names for tor hidden services. Based on eschalot.

Usage:
make 
./low-entropy-eschalot [min unique characters] [min repeating chains ]

example:
./low-entropy-eschalot 8 8
unique characters:8 
repeating chains:8
